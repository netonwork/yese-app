## 🔧 自定义配置

### 修改主色调
在 `src/styles/themes.css` 中修改CSS变量:
```css
.light {
  --primary-500: #your-color;
}
```

### 添加新分类
在 `HomePage.tsx` 中的 `mockCategories` 数组添加:
```typescript
{
  id: 'new-category',
  name: '新分类',
  icon: '🎯',
  count: 100,
  gradient: 'from-blue-500 to-green-500'
}
```
